Blackwood.IO Documentation

Welcome to the documentation for Blackwood.IO a .NET library that provides utilities for file operations, caching, and more.

Overview

Blackwood.IO is a set of high-level utilities for performing file and directory operations, caching, and abstracting common IO patterns. It simplifies several file-system tasks.

Getting Started

Prerequisites

  • .NET 8.0 or later
  • Visual Studio 2022 or later (Windows), or VS Code with C# extension (cross-platform)

Installation

Install the package via NuGet:

dotnet add package Blackwood.IO

Or using Package Manager Console:

Install-Package Blackwood.IO
